# a006: 一元二次方程式

標籤 : 數學

[內容]
求一元二次方程式 ax2+bx+c=0 的根

[輸入說明]
每組輸入共一行，內含三個整數 a, b, c 以空白隔開。

[輸出說明]

Two different roots x1=?? , x2=??

Two same roots x=??

No real root

PS: 答案均為整數，若有兩個根則大者在前


@解題提示：  
一元二次方程式要有實數解，判別式必須要大於零。  
即No real root的原由。  
  
判別式的結果，決定了-> 1. 是否有實數解 2. 是否為相異根解  
  
  
@解題提示：  
C引入數學函式-> math.c  
gcc 編譯時指令，功能選項要使用 -lm  
$ gcc a006.c -o a006 -lm
